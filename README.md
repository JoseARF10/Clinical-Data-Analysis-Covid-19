COVID-19 Comorbidity Correlation Analysis
Project Overview
This project analyzes a comprehensive COVID-19 patient dataset to investigate how pre-existing conditions—specifically diabetes and obesity—correlate with mortality outcomes.

Technical Stack

Language: Python

Libraries: Pandas (Data manipulation and analysis)

Dataset: https://www.kaggle.com/datasets/meirnizri/covid19-dataset

Installation & Execution

Dependencies: Install the required packages by running:
pip install -r requirements.txt

Usage: Execute the main script via the terminal.

Interaction: When prompted, enter 'diabetes', 'obesity', or 'exit' to generate the specific analysis or terminate the program.
